Users
=====

By `Brett Burbidge`_

Temporary User Accounts
^^^^^^^^^^^^^^^^^^^^^^^
Temporary user accounts are not permitted in UTA. 

Non-Repudiation Services
^^^^^^^^^^^^^^^^^^^^^^^^
Non-repudiation services are not defined as part of the application design.

Centralized Logging
^^^^^^^^^^^^^^^^^^^
UTA does not provide centralized logging. The user accept the risk for not utilizing a centralized logging solution.
